## Installing
`pip install -r requirements.txt`

## Usage
In `.env` File Added:
`BOT_TOKEN=your_bot_token`

Bot Prefix `#`  
eg:`#hello`

In `config.json` File Change the Setting

eg.
|  Key   |  Value  |
|  ----  | ----  |
| announcement  | announcements |
| prefix  | # |

## Description
Admin Guard is a Discord bot built with Python and discord.py, designed for server administration.
It supports admin commands while providing informational and fun commands for regular members.

## Requirements

- Python 3.13.7 or above

- Bot must have administrator permissions for full functionality

- You may customize channel and role names in the code as needed

- Make sure the .env file contains your correct BOT_TOKEN
